## How to Use This Skill

When working on product management tasks, I can help you with:

1. **Strategic Planning**: Define vision, set OKRs, prioritize roadmaps
2. **User Research**: Design interviews, analyze feedback, run usability tests
3. **Team Leadership**: Hire effectively, run 1:1s, build culture
4. **Growth**: Design growth loops, optimize retention, pricing strategy
5. **Execution**: Ship products, manage timelines, collaborate cross-functionally
6. **AI Products**: Strategy for AI/LLM products, evaluation frameworks
