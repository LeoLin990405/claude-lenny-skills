# Claude Lenny Skills

Lenny's Podcast product management skills for Claude Code - 86 actionable PM skills across hiring, strategy, growth, leadership, and more.

## Overview

This skill collection brings the wisdom from Lenny's Podcast into Claude Code, providing structured frameworks and actionable guidance for product managers.

## Skills Included

| Skill | Description |
|-------|-------------|
| `lenny-skills` | Master index of all 86 PM skills |
| `lenny-playbooks` | Role-based skill combinations |
| `lenny-strategy` | Vision, roadmaps, OKRs, PRDs |
| `lenny-execution` | Delivery, timelines, cross-functional collaboration |
| `lenny-decision` | Decision processes, uncertainty, trade-offs |
| `lenny-growth` | PMF, growth loops, pricing, retention |
| `lenny-research` | User research, interviews, surveys |
| `lenny-marketing` | Positioning, storytelling, launches, content |
| `lenny-hiring` | Job descriptions, interviews, onboarding, culture |
| `lenny-sales` | Founder sales, sales teams, enterprise, BD |
| `lenny-startup` | Team rituals, fundraising, pivoting |
| `lenny-design` | Design systems, reviews, engineering culture |
| `lenny-technology` | AI strategy, LLMs, platform, tech debt |
| `lenny-communication` | Presentations, writing, stakeholder alignment |
| `lenny-career` | Ideation, imposter syndrome, productivity |
| `lenny-advanced` | Product taste, systems thinking, behavioral design |

## Installation

```bash
cd ~/.claude/skills
git clone https://github.com/LeoLin990405/claude-lenny-skills.git
```

## Usage

```
/lenny-skills          # Browse all 86 skills
/lenny-playbooks       # Get role-based recommendations
/lenny-growth          # Access growth-specific skills
```
